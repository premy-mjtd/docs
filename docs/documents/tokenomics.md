# Tokenomics

!!! warning "This document is subject to change"
    Token details are finalized closer to the launch date. The information below reflects the current design.

## Overview

The Whaleden token powers the platform's creator economy. It aligns incentives between creators, fans, and the platform itself.

## Token Utility

| Use Case | Description |
|----------|-------------|
| **Governance** | Token holders vote on platform decisions |
| **Staking** | Stake tokens to earn yield and unlock premium features |
| **Creator Rewards** | Creators earn tokens based on community engagement |
| **Fan Rewards** | Fans earn tokens through referrals, purchases, and participation |
| **Access** | Certain content and features are gated by token holdings |

## Distribution

| Allocation | Percentage |
|------------|------------|
| Community & Ecosystem | 40% |
| Creator Rewards Pool | 25% |
| Team & Advisors | 15% |
| Treasury | 12% |
| Public Sale | 8% |

## Vesting

- **Team & Advisors** — 12-month cliff, 36-month linear vest
- **Creator Rewards Pool** — Released continuously based on platform activity
- **Community & Ecosystem** — Distributed over 5 years via platform incentives

## Points → Token Conversion

Points earned on the platform today will be convertible to tokens at a defined ratio upon token launch. Accumulate points now through:

- Referrals
- Comic purchases
- Supporting pitches
- Community participation
